msaf README
==================

msaf - MicroSatellite Analytics Framework

Getting Started
---------------

- cd <directory containing this file>

- install numpy & scipy
  $venv/bin/pip install numpy
  $venv/bin/pip install scipy

- install the rest of dependency
  $venv/bin/python setup.py develop

- $venv/bin/msaf-setup development.ini

- $venv/bin/pserve development.ini
